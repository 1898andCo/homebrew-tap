# 1898andCo Homebrew Tap

Homebrew formulae for tools by [1898andCo](https://github.com/1898andCo).

## Install

```bash
brew tap 1898andCo/tap
brew install tally
```

| Formula | Description |
|---------|-------------|
| `tally` | Git-backed findings tracker for AI coding agents |
